# Install

```bash
# clone the repo to your local
git clone https://github.com/zhentian-wan/MEANAppsFiles.git

# cd to the project dir
npm install
bower install
```

# Run

```bash
# Start the database
mongod

# Start the server
nodemon server
```

# View

Visit localhost:3030 to view the page.
